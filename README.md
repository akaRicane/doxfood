# Installation

## Packages

### mkcert (HTTPS certificates)

[HTTPS Certificates](https://github.com/FiloSottile/mkcert).

#### MacOs
```
brew install mkcert
```

#### Windows with [Scoop](https://scoop.sh/) in powershell
```
scoop bucket add extras
scoop install mkcert

cd {repository}
mkcert localhost
```
